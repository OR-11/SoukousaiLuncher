GlobalHookTest.exeは絶対にプロセスキルしてはいけない
キーフックを解除する必要があるため、必ずウィンドウを閉じることで終了させること。

使い方
～ゲーム側編～
①game.ysfjkをメモ帳か何かで開いて設定
例文(これをコピペして編集してください):{"type":"luncher_verify","icon":"[アイコンがあれば画像ファイル。拡張子はjpgかpng]","app":"[ゲーム名(拡張子含む)]","title":"[タイトル]","description":"[一言説明]","enable":"true"}
書き方はJSON形式です。
	type:
		さわらないでください。要するによく考えたら実装する必要がなかったおまじないです。

	icon:
		アイコン画像ファイル。pngかjpgです。設定したパスが無効(空欄や存在しないパス等)な場合、アイコンはランチャーに内蔵されている代用品になります。

	app:
		ゲームファイル。そのPCで実行可能ならば、exeだけでなくjarやbat、txt、htmlなど、すべてのファイルを開くことが可能です。

	title:
		タイトル。少し長くても文字サイズが自動的に調整されますが、長すぎないように。

	discription:
		長すぎるとUIが崩れるので、一言で。

	enable:
		trueの場合は通常通り遊べ、falseの場合はメニューに表示はされますが起動することはできなくなります。

②完了。プレイ中はゲームウィンドウがランチャーの上に表示され、最小化はできなくなります。

ゲームの操作してほしいウィンドウ(=その時点で入力を受け付けるウィンドウ)がゲーム起動時に実行するアプリのものでない場合、もしくは途中でウィンドウが変わる場合、ゲームの実行ファイルが置かれているディレクトリに、下記の書式で書いたテキストファイルを"MainWindowHandle.ysfjk"という名前で保存してください。
また、ゲームをプレイ中に実行されるプロセスが最初に起動されるプロセス以外にもある場合もMainWindowHandle.ysfjkを利用してください。
MainWindowHandle.ysfjkを更新/作際したら、同じディレクトリに"ChangeMainWindow.ysfjk"という空のファイルを作成してください。ChangeMainWindow.ysfjkが存在する間はMainWindowHandle.ysfjkを閉じていてください。
ウィンドウが変わらない人はそのまま何もしなくて大丈夫です

MainWindowHandle.ysfjkの書式:
MainWindowHandle=XXX
ProcessIds=XXX,XXX,XXX,...

	MainWindowHandle:
		入力を与えるべきウィンドウ(最上位に表示されるウィンドウ)のWindowHandleを、10進数で書いてください。

	ProcessIds:
		ゲーム終了時にもし閉じていなかったら閉じるべきプロセスのIdを、,区切りで羅列してください。もしすでに終了していたなどの原因で終了に失敗した場合、Logファイルにエラーメッセージを出力する...はずです。

※注意※
Altキー、F4キーはおそらく使えませんが、ゲーム起動中にAlt+F4が押された場合はランチャーがゲームを終了させます。

～ランチャー編～
①bootsetting.ysfjkをメモ帳か何かで開いて設定
例文(これをコピペして編集してください):{"online":"false","GamePath":[],"AdditionalPathToExportLogFile":[],"GetNumber":"","PostURL":""}
書き方はJSON形式です。
	online:
		true...オンラインモード※プロキシ関係がめんどいので完全に実装してません。
		false...オフラインモード
		prefer...オンラインモードで起動し、接続に失敗した場合はオフラインモードに移行
	※「online」とその値はダブルクオーテーションで囲うこと。

	GamePath:
		ローカル環境から接続できる場所にあるゲームフォルダのパスを、絶対パスで書いてください。
		全体を[]で囲み、パスを一つずつダブルクオーテーションで囲み、パスが2つ以上あるときは,で区切ってください。

	AdditionalPathToExportLogFile:
		もし、このランチャーのディレクトリ以外の場所にもログファイルを出力したい場合、ここに『絶対パス』で書いてください。
		全体を[]で囲み、パスを一つずつダブルクオーテーションで囲み、パスが2つ以上あるときは,で区切ってください。

②SoukousaiGameLuncher.exeを起動してください。

③終了する際、オフラインモードの時はCtrl+Alt+Delでタスクマネージャーを開き、SoukousaiGameLuncher.exeを終了してください。GlobalHookTest.exeはSoukousaiGameLuncher.exeが終了しても閉じないため、手動でウィンドウを閉じてください。その後タスクマネージャーをアクティブにしたまWindowsキーを押し設定を開き、個人用設定→タスクバー→タスクバーの動作で「タスクバーを自動的に隠す」に一度チェックを入れ外してください。タスクバーが復活します。
